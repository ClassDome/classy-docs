# Classy

Classy is a frontend library made by ClassDome to aid in their frontend development needs.

## Getting Started

To get started you should install Classy

<Tabs>
<TabItem value="yarn" label="Yarn">

```bash
yarn add @classdome/classy
```

</TabItem>
<TabItem value="npm" label="NPM">

```bash
npm install --save @classdome/classy
```

</TabItem>
<TabItem value="pnmp" label="PNPM">

```bash
pnpm add @classdome/classy
```

</TabItem>
</Tabs>

Or **try Classy** in your browser with `Soon™`

### Requirements

- [Node.js](https://nodejs.org/en/download/) version 16.14 or above:
- One of the following:
  - [Yarn](https://yarnpkg.com/) verison 3.1.0 or higher
  - [NPM](https://npmjs.com/) version 9.2.0 or higher
  - [PNPM](https://pnpm.io/) version 7.10.0 or higher
- `react` and `react-dom` version 17.0.2 or higher
